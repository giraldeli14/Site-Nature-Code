# 🌿 Nature Site - Portal Educacional de Biologia

Um site educacional interativo desenvolvido para o estudo de biologia, abordando três grandes módulos: **Ecossistemas**, **Reino Animal** e **Reino Plantae**.

## 📋 Sobre o Projeto

Este projeto foi desenvolvido como trabalho acadêmico com o objetivo de criar uma plataforma digital para o ensino de biologia, oferecendo conteúdo didático estruturado e quizzes interativos para fixação do aprendizado.

## 🎯 Funcionalidades

- **Conteúdo Educacional Estruturado**: Organizado em módulos e tópicos temáticos
- **Interface Responsiva**: Adaptável para diferentes dispositivos
- **Quizzes Interativos**: Testes de conhecimento para cada tópico estudado
- **Navegação Intuitiva**: Sistema de módulos e subtópicos para fácil acesso ao conteúdo
- **Design Moderno**: Interface visual atraente com imagens ilustrativas

## 🚀 Como Usar

1. **Clone o repositório** ou faça o download dos arquivos
2. **Abra o arquivo `index.html`** no seu navegador
3. **Navegue pelos módulos** disponíveis na página inicial
4. **Explore os tópicos** dentro de cada módulo
5. **Faça os quizzes** para testar seus conhecimentos

## 💻 Tecnologias Utilizadas

- **HTML5**: Estruturação do conteúdo
- **CSS3**: Estilização e design responsivo
- **JavaScript**: Interatividade e quizzes
- **LocalStorage**: Armazenamento de progresso dos quizzes

## 📱 Responsividade

O site foi desenvolvido com design responsivo, adaptando-se automaticamente a diferentes tamanhos de tela:
- Desktop
- Tablet
- Mobile
